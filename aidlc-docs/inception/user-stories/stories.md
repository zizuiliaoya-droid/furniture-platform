# 用户故事

按功能模块组织，细粒度，Given-When-Then 验收标准。

---

## 模块 1：认证与登录

### US-1.1 登录页台灯交互
**作为** 任何访问者，**我想要** 在登录页看到一个可交互的台灯动画，**以便** 获得有趣的视觉体验，同时引导我注意到登录表单。

**验收标准：**
- Given 用户访问登录页，When 页面加载完成，Then 显示暗色背景（#121921）和台灯 SVG，登录表单默认隐藏（opacity: 0）
- Given 台灯处于关闭状态，When 用户拖拽拉绳超过 50px，Then 台灯开启，灯光以随机 hue 值亮起
- Given 台灯刚开启，When 灯光亮起，Then 登录表单以弹簧动画（cubic-bezier 弹性曲线）淡入并放大出现
- Given 台灯开启状态，When 用户再次拖拽拉绳超过 50px，Then 台灯关闭，登录表单淡出隐藏
- Given 台灯开启，When 观察台灯眼睛，Then 眼睛朝上（rotate 0°）；Given 台灯关闭，Then 眼睛朝下（rotate 180°）
- Given 台灯开启，When 观察登录表单，Then 表单边框和阴影颜色与台灯当前灯光颜色一致
- Given 拉绳被拖拽，When 释放拉绳，Then 绳子以 MorphSVGPlugin 弹动动画回弹
- Given 页面在移动端，When 屏幕宽度不足，Then 台灯和表单纵向排列（响应式换行）

**关联画像：** 李总、小王、张经理（登录页对所有人可见）

---

### US-1.2 用户登录
**作为** 内部用户（管理员/员工），**我想要** 使用用户名和密码登录系统，**以便** 访问内部管理功能。

**验收标准：**
- Given 台灯已开启且登录表单可见，When 用户输入正确的用户名和密码并点击登录，Then 系统返回 Token 并跳转到首页仪表盘
- Given 用户输入错误的用户名或密码，When 点击登录，Then 显示错误提示"用户名或密码错误"
- Given 用户账号被禁用（is_active=false），When 尝试登录，Then 显示提示"账号已被禁用，请联系管理员"
- Given 登录成功，When Token 存储到 localStorage，Then 后续所有 API 请求自动携带 Authorization: Token {token}
- Given 输入框获得焦点，When 用户点击输入框，Then 显示彩色发光边框效果

**关联画像：** 李总、小王

---

### US-1.3 用户登出
**作为** 已登录用户，**我想要** 安全登出系统，**以便** 保护我的账号安全。

**验收标准：**
- Given 用户已登录，When 点击登出按钮，Then 调用登出 API 删除服务端 Token
- Given 登出成功，When Token 被删除，Then 清除 localStorage 中的 Token 并跳转到登录页
- Given API 返回 401，When 任何请求收到未授权响应，Then 自动清除 Token 并跳转登录页

**关联画像：** 李总、小王

---

### US-1.4 用户管理
**作为** 管理员，**我想要** 创建、编辑和管理用户账号，**以便** 控制谁可以访问系统。

**验收标准：**
- Given 管理员在用户管理页面，When 查看用户列表，Then 显示所有用户的用户名、显示名称、角色、状态
- Given 管理员点击"创建用户"，When 填写用户名、密码、显示名称、角色并提交，Then 新用户创建成功
- Given 管理员编辑用户信息，When 修改显示名称或角色并保存，Then 用户信息更新成功
- Given 管理员点击"禁用"按钮，When 确认操作，Then 该用户状态变为禁用，无法登录
- Given 管理员点击"启用"按钮，When 确认操作，Then 该用户状态恢复启用
- Given 创建用户时用户名已存在，When 提交表单，Then 显示错误提示"用户名已存在"

**关联画像：** 李总

### US-1.5 密码重置
**作为** 员工，**我想要** 在忘记密码时联系管理员重置密码，**以便** 恢复系统访问。

**验收标准：**
- Given 用户在登录页，When 点击"忘记密码？"链接，Then 显示提示"请联系管理员重置密码"
- Given 管理员在用户管理页面，When 点击某用户的"重置密码"按钮，Then 显示密码重置表单
- Given 管理员输入新密码，When 提交重置，Then 该用户密码更新成功，下次登录使用新密码

**关联画像：** 小王、李总

### US-1.6 权限控制
**作为** 系统，**我想要** 根据用户角色限制功能访问，**以便** 保护数据安全。

**验收标准：**
- Given 员工（STAFF）登录，When 访问产品管理页面，Then 只能查看产品，不显示新建/编辑/删除按钮
- Given 管理员（ADMIN）登录，When 访问产品管理页面，Then 显示完整的新建/编辑/删除功能
- Given 员工尝试通过 API 直接调用管理员接口，When 发送请求，Then 返回 403 Forbidden
- Given 未登录用户，When 访问受保护路由，Then 自动跳转到登录页

**关联画像：** 李总、小王

**作为** 管理员，**我想要** 创建新产品并填写详细信息，**以便** 将产品录入系统供团队使用。

**验收标准：**
- Given 管理员在产品管理页面，When 点击"新建产品"，Then 跳转到产品创建表单页
- Given 产品创建表单，When 填写名称、编号、产地、描述、最低售价、主分类并提交，Then 产品创建成功并跳转到产品详情页
- Given 产品编号已存在，When 提交表单，Then 显示错误提示"产品编号已存在"
- Given 必填字段未填写，When 提交表单，Then 显示对应字段的验证错误提示
- Given 产品创建成功，When 查看产品详情，Then 显示所有填写的信息，默认状态为上架

**关联画像：** 李总

**作为** 管理员，**我想要** 编辑已有产品的信息，**以便** 保持产品数据的准确性。

**验收标准：**
- Given 管理员在产品详情页，When 点击"编辑"按钮，Then 跳转到产品编辑表单，表单预填当前数据
- Given 编辑表单中修改了产品信息，When 提交保存，Then 产品信息更新成功，updated_at 时间更新
- Given 编辑时将编号改为已存在的编号，When 提交，Then 显示错误提示

- Given 确认删除，When 用户点击"确定"，Then 产品及其关联的图片、配置被删除
- Given 产品被报价单引用，When 删除产品，Then 报价单中的产品名称快照保留不受影响

**验收标准：**
- Given 用户在产品列表页，When 页面加载，Then 显示分页的产品列表（默认每页 20 条）
- Given 产品列表，When 输入搜索关键词，Then 同时匹配名称、编号、描述、配置名称、配置属性
- Given 产品列表，When 选择产地筛选（进口/国产/定制），Then 只显示对应产地的产品
- Given 产品列表，When 选择分类筛选，Then 显示该分类及其子分类下的所有产品
- Given 产品列表，When 管理员切换"上架/下架"筛选，Then 按上架状态过滤产品
- Given 员工查看产品列表，When 页面加载，Then 只显示上架状态的产品

- Given 产品详情页，When 查看图片区域，Then 显示所有产品图片，封面图优先展示
- Given 产品详情页，When 查看底部链接区域，Then 显示设计资源、培训资料、资质文件的快捷跳转链接

**作为** 管理员，**我想要** 控制产品的上架/下架状态，**以便** 管理员工可见的产品范围。

**验收标准：**
- Given 管理员在产品详情页或列表页，When 点击"下架"按钮，Then 产品 is_active 设为 false，员工不再可见

### US-2.7 分类管理
**作为** 管理员，**我想要** 管理三维度的树形分类体系，**以便** 组织产品分类结构。

**验收标准：**
- Given 管理员在分类管理页面，When 选择维度（类型/空间/产地），Then 显示该维度的树形分类结构
- Given 分类树，When 点击"新建分类"，Then 可创建顶级分类或子分类
- Given 分类树，When 拖拽分类项，Then 调整排列顺序，调用批量更新排序接口
- Given 编辑分类，When 修改分类名称并保存，Then 分类名称更新成功
- Given 删除分类，When 分类下无产品和子分类，Then 删除成功
- Given 删除分类，When 分类下有产品或子分类，Then 显示错误提示，不允许删除
- Given 同一父分类下，When 创建同名分类，Then 显示错误提示"同级分类名称不能重复"

## 模块 5：内部文档

### US-5.1 文件夹管理
**作为** 管理员，**我想要** 创建和管理文件夹结构，**以便** 组织内部文档。

**验收标准：**

**验收标准：**

- Given 分享链接已被禁用，When 客户打开链接，Then 显示"链接已失效"提示
