# 用户画像

## Persona 1：管理员 — 李总（ADMIN）

| 属性 | 描述 |
|------|------|
| **角色** | 公司管理层 / 运营总监 |
| **年龄** | 35-50 岁 |
| **技术水平** | 中等，熟悉基本办公软件，不熟悉技术细节 |
| **使用频率** | 每天使用，主要在工作时间 |
| **设备** | 台式电脑（办公室）、笔记本（出差） |
| **核心目标** | 高效管理公司产品资料、案例、报价，快速响应客户需求 |
| **痛点** | 产品资料分散在多个 Excel/文件夹中，查找困难；报价制作耗时；无法追踪分享给客户的资料是否被查看 |
| **期望** | 一站式管理所有资料，快速生成专业报价单，了解客户对分享内容的关注度 |
| **权限** | 全部权限：增删改查所有数据、用户管理、系统配置 |

### 典型使用场景
1. 录入新产品信息和图片，设置多配置价格
2. 创建报价单发送给客户，追踪客户查看情况
3. 管理员工账号，控制数据访问权限
4. 查看仪表盘了解业务概况

## Persona 2：普通员工 — 小王（STAFF）

| 属性 | 描述 |
|------|------|
| **角色** | 销售顾问 / 设计师 |
| **年龄** | 25-35 岁 |
| **技术水平** | 较高，熟悉各类办公和设计软件 |
| **使用频率** | 每天高频使用 |
| **设备** | 笔记本电脑、偶尔使用平板展示给客户 |
| **核心目标** | 快速查找产品信息，制作报价方案，分享资料给客户 |
| **痛点** | 需要频繁翻阅产品目录查找信息；手动制作报价单容易出错；无法快速给客户展示案例 |
| **期望** | 快速搜索定位产品，一键生成报价，便捷分享产品和案例给客户 |
| **权限** | 查看所有数据、创建报价单和分享链接，不能删除数据或管理用户 |

### 典型使用场景
1. 接待客户时搜索产品，展示产品图册和案例
2. 根据客户需求创建报价单，选择产品配置
3. 生成分享链接发送给客户，方便客户回去后查看
4. 查阅培训资料和设计资源

## Persona 3：外部客户 — 张经理

| 属性 | 描述 |
|------|------|
| **角色** | 企业采购负责人 / 项目经理 |
| **年龄** | 30-50 岁 |
| **技术水平** | 基础，主要使用微信和浏览器 |
| **使用频率** | 偶尔，收到分享链接时访问 |
| **设备** | 手机（微信打开链接）、电脑 |
| **核心目标** | 查看供应商发来的产品信息和报价，做采购决策 |
| **痛点** | 收到的产品资料格式不统一，难以比较；报价单不够直观 |
| **期望** | 打开链接即可查看专业的产品展示和清晰的报价明细 |
| **权限** | 仅通过分享链接访问特定内容，无需登录 |

### 典型使用场景
1. 收到销售发来的分享链接，在微信/浏览器中打开查看
2. 浏览产品图片和详细信息，了解不同配置和价格
3. 查看报价单明细，确认价格和折扣
4. 多次回看分享内容，做采购决策参考
