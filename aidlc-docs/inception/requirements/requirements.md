# 家具软装内部管理平台 — 需求分析文档

## 意图分析

| 项目 | 内容 |
|------|------|
| **用户请求** | 按照提供的开发文档和需求文档开发家具软装内部管理平台，测试环境采用 Railway+Vercel，测试通过后部署到用户私有云（Synology NAS），登录页采用暗色台灯交互设计 |

---

## 1. 项目概述

### 1.1 项目背景
家具软装公司内部资料管理系统，管理产品信息、客户案例、内部文档、报价方案，支持通过分享链接将信息发送给外部客户查看。

### 1.2 目标用户
- **管理员（ADMIN）**：公司管理层，全部权限
- **普通员工（STAFF）**：销售/设计人员，查看+部分创建权限
- **外部客户**：通过分享链接访问，无需登录

### 1.3 仓库结构
- **双仓库（Dual Repo）**：前端和后端分开两个独立 GitHub 仓库
- 前端仓库：React + TypeScript 项目
- 后端仓库：Django + DRF 项目

## 3. 功能需求

### 3.1 登录页（暗色台灯交互设计）

| 需求项 | 说明 |
|--------|------|
| 页面主题 | 暗色主题，背景色 #121921 |
| 台灯 SVG | 页面中央可交互台灯装饰 |
| 拉绳交互 | GSAP Draggable 实现，拖拽超过 50px 触发开关 |
| 绳子动画 | GSAP MorphSVGPlugin 制作弹动动画 |
| 音效 | **不需要音效**（用户明确要求） |
| 灯光变色 | 切换时随机 hue 值，CSS 变量 --on 和 --shade-hue 联动 |
| 台灯眼睛 | 关闭时朝下（rotate 180°），开启时朝上（rotate 0°） |
| 登录表单 | 默认隐藏，台灯开启时弹簧动画淡入 |
| 表单样式 | 边框和阴影使用台灯当前随机颜色 |
| 表单内容 | 标题"欢迎回来"、账号输入框、密码输入框、登录按钮、"忘记密码？"链接 |
| 输入框交互 | 获焦时显示彩色发光边框 |
| 布局 | 台灯和表单横向并排居中，flex 布局，响应式换行 |

### 3.2 用户认证与权限

| 功能 | 说明 |
|------|------|
| 登录/登出 | 用户名+密码，Token 认证 |

## 5. 部署需求

### 5.1 测试环境（Railway + Vercel）

| 项目 | 说明 |
|------|------|
| 前端 | Vercel 自动部署，CDN 加速 |
| 后端 | Railway（Django + Gunicorn + PostgreSQL） |
| CI/CD | GitHub main 分支推送自动部署 |
| 静态文件 | Vercel 托管前端，Railway WhiteNoise 服务 Django 静态文件 |
| 媒体文件 | Railway 持久化存储卷 |

### 5.2 生产环境（Synology NAS）

| 项目 | 说明 |
|------|------|
| 硬件 | Synology DS920+ NAS |
| 部署方式 | Docker Compose 一键部署（db + backend + frontend） |
| 访问方式 | 局域网直接访问 + Synology QuickConnect 外网访问 |
| 外网需求 | 分享功能需要外网访问（外部客户通过 QuickConnect 访问分享链接） |
| 自启动 | Docker 容器 restart: always |
| 数据备份 | NAS Hyper Backup |
| 前端服务 | Nginx（静态文件 + API 反向代理 + 媒体文件） |
| 后端服务 | Gunicorn |

## 7. 关键决策记录

| # | 决策 | 原因 |
|---|------|------|
| 1 | GSAP 商业许可 | 计划购买 Club GreenSock 会员，使用 MorphSVGPlugin |
| 2 | 不需要音效 | 用户明确要求登录页台灯交互不需要咔哒音效 |
| 3 | WeasyPrint + Dockerfile 系统依赖 | 测试和生产环境都安装系统依赖，保持一致性 |
| 4 | Excel 模板按 Product 模型字段 | 名称、编号、产地、描述、最低售价、分类 |
| 5 | 字体自托管 | NAS 可能无外网，字体文件打包到项目中 |
| 6 | 分享页与主系统风格一致 | 保持品牌统一性 |
| 7 | QuickConnect 外网访问 | 分享功能需要外部客户访问 |
| 8 | 首页为仪表盘概览 | 展示统计信息和快捷入口 |
| 9 | 双仓库结构 | 前后端独立仓库，独立部署 |
| 10 | UI UX Pro Max 设计系统 | 使用 AI 设计智能 skill 生成前端设计系统 |
