# 家具软装内部管理平台 — 前端

React + TypeScript + Ant Design 前端应用。

## 技术栈
- React 18 + TypeScript 5
- Ant Design 5.x (包豪斯主题)
- Vite 5.x
- Zustand (状态管理)
- GSAP (登录页动画)

## 本地开发
```bash
npm install
npm run dev
```

## 环境变量
- `VITE_API_BASE_URL`: 后端 API 地址（Vercel 部署时设置为 Railway URL）

## 构建
```bash
npm run build
```
